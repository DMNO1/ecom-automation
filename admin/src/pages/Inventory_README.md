# 库存管理页面

## 功能概述

创建了一个完整的库存管理页面，用于监控和管理电商平台的库存情况。

## 主要功能

### 1. 统计概览
- **总SKU数**: 显示所有商品的SKU总数
- **低库存商品**: 库存低于补货点的商品数量
- **缺货商品**: 库存为0的商品数量
- **库存总价值**: 所有商品库存的总价值

### 2. 库存预警
- 当有低库存或缺货商品时，显示醒目的预警横幅
- 点击"查看低库存商品"可快速筛选显示需要补货的商品

### 3. 筛选功能
- **搜索**: 支持按SKU或商品名称搜索
- **分类筛选**: 按商品分类筛选（电子产品、配件、服装等）
- **状态筛选**: 按库存状态筛选（低库存、缺货、正常）

### 4. 库存表格
显示以下列信息：
- SKU（可复制）
- 商品名称
- 分类（标签显示）
- 当前库存（带进度条显示库存水平）
- 补货点（建议的最低库存量）
- 状态（低库存/缺货/正常，颜色标识）
- 单价
- 库存价值（当前库存×单价）
- 供应商
- 最后补货日期
- 操作按钮（补货、编辑、详情）

### 5. 补货功能
- 点击"补货"按钮打开补货申请弹窗
- 自动计算建议补货数量（补货点 - 当前库存）
- 可调整补货数量
- 提交后显示成功消息

## 数据说明

使用了15条mock数据，包含：
- 电子产品类：无线蓝牙耳机、智能手表、充电宝
- 配件类：手机壳、数据线
- 服装类：运动鞋、速干运动裤
- 生活用品：保温杯
- 箱包类：背包
- 配饰类：墨镜
- 健康类：按摩枕、体重秤
- 运动类：瑜伽垫
- 美妆类：防晒喷雾、护手霜套装

## 样式特点

采用深色主题设计，与现有仪表盘保持一致：
- 深色背景（#12131a）
- 渐变色统计卡片
- 低库存行高亮显示（橙色背景）
- 缺货行高亮显示（红色背景）
- 专业的表格样式和交互效果

## 后续扩展建议

1. **数据导出**: 支持导出库存数据为Excel/CSV格式
2. **批量操作**: 支持批量补货、批量修改补货点
3. **库存预警设置**: 允许自定义每个商品的补货点
4. **供应商管理**: 集成供应商信息和联系方式
5. **库存历史**: 查看库存变化历史记录
6. **自动补货**: 设置自动补货规则，当库存低于补货点时自动下单
7. **库存预测**: 基于销售数据预测未来库存需求
